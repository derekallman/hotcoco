# Security Policy

## Supported Versions

Security fixes are applied to the latest release only.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security concerns by emailing the maintainer via the contact listed on the [GitHub profile](https://github.com/derekallman), or open a [GitHub private security advisory](https://github.com/derekallman/hotcoco/security/advisories/new).

We will acknowledge the report within 48 hours and aim to release a patch within 7 days for confirmed vulnerabilities.
